# 🖥️ CPU Scheduling Simulator

An interactive web-based visualization tool for understanding CPU scheduling algorithms in Operating Systems.

## ✨ Features

- **3 Scheduling Algorithms:** FCFS, SRTF, MLFQ
- **Dynamic Gantt Charts** with color-coded process visualization
- **Step-by-Step Mode** with animated walkthrough
- **Real-time Metrics:** Average Waiting Time, Turnaround Time, CPU Utilization, Throughput
- **MLFQ Customization:** Per-queue algorithms (RR/FCFS/SJF/SRTF) and aging threshold
- **Process Analysis Table** with execution segment breakdown
- **Responsive Design** — Works on all devices

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS Custom Properties, Flexbox)
- Vanilla JavaScript
- Canvas API

## 📊 Algorithms Explained

| Algorithm | Type | Description |
|-----------|------|-------------|
| **FCFS** | Non-preemptive | First Come First Served - simple queue-based execution |
| **SRTF** | Preemptive | Shortest Remaining Time First - optimal average waiting time |
| **MLFQ** | Adaptive | Multilevel Feedback Queue - used in modern operating systems |
